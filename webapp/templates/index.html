Welcome to PyMiniDB

PyMiniDB is a minimal relational database management system built from scratch in Python. This web interface allows you to interact with the database through a user-friendly interface.

Features:
  • Create and manage database tables
  • Execute SQL queries
  • View and edit table data
  • Task management example (full CRUD operations)
  • Basic indexing support
